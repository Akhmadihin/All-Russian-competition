# All-Russian-competition
# Программа для изучения английских слов

Описание проекта

Данная программа поможет в изучении английских слов и неправильных глаголов. Особенно актуально для подготовки к ОГЭ и ЕГЭ. Приложение позволяет создавать собственные словари, тренировать их, а также прослушивать правильное произношение.

Функционал

- Работа с пользовательскими словарями - возможность создавать, редактировать и учить слова из собственных текстовых файлов
- Неправильные глаголы - полный список глаголов, необходимых для сдачи экзаменов
- Озвучивание слов - с помощью голосового движка edge-tts
- Два режима ввода:
  - Ручной ввод перевода
  - Автоматический перевод через встроенный переводчик
- Система запоминания - слова, в которых допущены ошибки, повторяются до полного усвоения
- Настройка режимов работы

Установка

Требования
- Python 3.7+
- Установка зависимостей:

```
pip install edge-tts pygame translate
```

Структура файлов

```
├── config.txt                 # Настройки программы
├── Irregular_listen.txt       # Глаголы для аудирования
├── Irregular_learn.txt        # Глаголы для изучения (с переводом)
├── word.txt                   # Пример пользовательского словаря
└── конкурс Всероссийский.py   # Основной исполняемый файл
```

Запуск

```
python "конкурс Всероссийский.py"
```

Меню программы

1. Учить свои слова - запуск тренировки по вашему словарю
2. Учить неправильные глаголы - вывод списка глаголов с переводом и транскрипцией
3. Прослушать неправильные глаголы - последовательное озвучивание глаголов
4. Ввести свои слова - создание или редактирование словаря
5. Прослушать свои слова - озвучивание введённых слов
6. Настройки - выбор режима ввода (ручной/автоматический)

## 📖 Формат словаря

Слова должны быть записаны в формате:
```
слово - перевод
word - слово
study - учиться, изучать
```

Озвучивание

Используется голос `en-GB-SoniaNeural` от edge-tts. Для воспроизведения требуется подключение к интернету.

Настройки

В файле `config.txt` хранится выбранный режим:
- `1` - ручной ввод перевода
- `2` - автоматический перевод

При первом запуске программа предложит выбрать режим.

Алгоритм тренировки

1. Слова перемешиваются случайным образом
2. Пользователь вводит перевод
3. В случае ошибки слово добавляется в список на повторение
4. Цикл повторяется, пока все слова не будут отвечены правильно

Для подготовки к экзаменам

В файлах `Irregular_learn.txt` и `Irregular_listen.txt` содержатся все неправильные глаголы, соответствующие кодификаторам ОГЭ и ЕГЭ. Каждый глагол сопровождается:
- Переводом на русский
- Инфинитивом
- Формами прошедшего времени
- Транскрипцией

Примечания

- Все текстовые файлы должны быть в кодировке UTF-8
- Для работы аудио необходим доступ в интернет
- При создании новых словарей рекомендуется указывать имя файла латиницей
